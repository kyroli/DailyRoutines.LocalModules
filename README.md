# DailyRoutines Local Modules

个人自用的 DailyRoutines (DR) 本地模块备份。

## 模块列表

### 1. 自动任务说话 (AutoQuestSay.cs)
点击任务目标自动发送指定台词。

<img src="images/aqs_icon.png" width="600" />
<br>
<img src="images/aqs_preview.png" width="600" />

### 2. 天书连线概率 (WondrousTailsPredictor.cs)
在天书界面显示连线概率及重排期望。

<img src="images/wt_icon.png" width="600" />
<br>
<img src="images/wt_preview.png" width="600" />

### 3. 自动每周仙人仙彩(改) (AutoJumboCactpotCustom.cs)
基于官方同名模块修改，自动购买并选择每周仙人仙彩号码。<br>
※ 增加了“一号多买”模式：首张票随机生成，后续票自动沿用该号码。

<img src="images/ajc_icon.png" width="600" />

### 4. 自动雇员作业(改) (AutoRetainerWorkCustom.cs)
基于官方同名模块修改，自动收取并重新派遣雇员。<br>
※ 增加了执行期间会自动开启“跳过对话”模块的功能。

<img src="images/arwc_icon.png" width="600" />
